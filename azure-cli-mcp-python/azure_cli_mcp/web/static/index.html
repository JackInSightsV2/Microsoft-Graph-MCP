Azure CLI MCP Server

Welcome to Azure CLI MCP Server - Python Version!

This is the Python implementation of the Azure CLI MCP Server, converted from the original Java version.

Features:

  • 🚀 MCP Integration: Full MCP (Model Context Protocol) server implementation
  • ⚡ Async Azure CLI: Asynchronous Azure CLI command execution
  • 🔐 Authentication: Support for device code and service principal authentication
  • 🛡️ Security: Input validation and command sanitization
  • 📝 Logging: Comprehensive logging and error handling
  • 🌐 Web Interface: FastAPI-based web interface

Available Endpoints:

  • Health Check - Service status and Azure CLI availability
  • API Info - Server information and available tools

To learn more about this server, visit the GitHub repository.

Original Java version: azure-cli-mcp
